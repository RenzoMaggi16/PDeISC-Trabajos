Acerca de este proyecto

¿Qué hace este proyecto?

Esta aplicación opera como un laboratorio visual (con un layout de grilado similar a un entorno de desarrollo IDE) para demostrar y materializar la manipulación directa de la arquitectura fundamental web: el DOM (Document Object Model). El usuario instanciará objetos estructurales y los someterá a mutaciones, mientras un observador cíclico lo reporta todo.

Conceptos de DOM que se practican

  • document.createElement() para levantar estructuras de la nada.
  • element.setAttribute() para forzar perfiles de hipervínculos (evitando properties).
  • element.getAttribute() para recuperar memoria previa a la sobre-escritura.
  • element.textContent para alterar strings textuales interiores.
  • Renderización visual asíncrona mediante un terminal (Audit Log).

Tecnologías utilizadas

  • Node.js y Express: Servidor asíncrono subyacente.
  • Vanilla JS (ES6): La lógica pura del motor. Ningún framework tipo React interviene la memoria RAM en el DOM.
  • Bootstrap 5 y CSS3: Para el diseño adaptativo.
