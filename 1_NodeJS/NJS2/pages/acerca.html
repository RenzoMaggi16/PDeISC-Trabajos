DESCRIPCIÓN

NJ2 es un proyecto educativo de Node.js organizado en cinco ejercicios progresivos. Cada ejercicio introduce un concepto nuevo y se apoya en los anteriores, culminando en un sitio web completo servido por un servidor HTTP nativo de Node.js.

El proyecto demuestra cómo construir una aplicación modular sin frameworks: módulos personalizados con lógica de negocio, módulos nativos de Node.js para funciones del sistema, paquetes de NPM para funcionalidades externas, y un servidor HTTP que sirve páginas HTML dinámicas mediante inyección de contenido con String.prototype.replace().

Todo el código está escrito en JavaScript usando CommonJS (require / module.exports) y puede ejecutarse con node server.js sin ninguna dependencia de transpilación ni bundler.

TECNOLOGÍAS UTILIZADAS

TECNOLOGÍA VERSIÓN USO EN EL PROYECTO
Node.js 18+ Entorno de ejecución y servidor HTTP con node:http
HTML5 — Estructura de todas las páginas del sitio web
Bootstrap 5 5.3.3 Estilos, grilla responsiva, tablas y componentes visuales
NPM — Gestión de paquetes externos del proyecto
upper-case 2.x Conversión de texto a mayúsculas para títulos y encabezados
CommonJS — Sistema de módulos con require y module.exports

LOS CINCO EJERCICIOS

# NOMBRE CONCEPTO DEMOSTRADO
01 Módulos personalizados Creación y exportación de módulos propios con module.exports: timeModule, calcModule, stringModule
02 HTTP y FS nativos Uso de los módulos nativos node:http y node:fs para crear servidores y leer archivos del disco
03 Módulo URL Análisis y construcción de URLs con el módulo nativo node:url: protocolo, host, path, query params y hash
04 Paquete NPM Instalación y uso del paquete upper-case de NPM para conversión de texto a mayúsculas
05 Sitio web completo Servidor HTTP que sirve múltiples páginas HTML dinámicas con inyección de contenido desde los módulos personalizados
