Metodo map() — Transformar y obtener nuevo array

map() aplica una funcion a cada elemento y devuelve un NUEVO array. El original no se modifica.

1. Multiplicar por 3

Multiplica cada numero por 3 y retorna un array nuevo.

2. Nombres en mayusculas

Convierte cada nombre en minúscula a mayusculas.

3. Precios con IVA (21%)

Calcula el precio final con impuesto para cada elemento.
