Metodo sort() — Ordenar arrays

sort() ordena el array original. Para numeros siempre se debe usar una funcion comparadora.

1. Numeros

Ordena los numeros usando una funcion comparadora (a - b) o (b - a).

2. Palabras alfabeticamente

Ordena textos usando localeCompare para prevenir errores.

3. Objetos por edad

Ordena un array de objetos comparando una propiedad especifica.
