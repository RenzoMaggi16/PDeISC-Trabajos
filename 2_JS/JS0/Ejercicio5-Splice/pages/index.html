splice()

Metodo splice() — Eliminar, insertar o reemplazar

splice(inicio, cantidad, ...nuevos) modifica el array en una posicion especifica.

1. Eliminar desde posicion

Se eliminaran 2 elementos a partir de la posicion 1 (indice 1).

2. Insertar sin eliminar

Inserta un elemento en el indice 1 sin borrar ningun elemento (cantidad a borrar = 0).

3. Reemplazar elementos

Se reemplazaran 2 elementos a partir del indice 2 con elementos nuevos.
