Metodo push() — Agregar al final

push() agrega uno o mas elementos al final de un array y devuelve la nueva longitud.

1. Frutas

Agrega "Manzana", "Banana" y "Naranja" uno por uno.

2. Amigos

Escribe el nombre de un amigo para agregarlo a la lista.

3. Numero condicional

Agrega el numero al array solo si es mayor que el ultimo elemento.
