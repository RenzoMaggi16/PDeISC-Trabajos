Metodo indexOf() — Buscar posicion

indexOf() devuelve el indice del elemento buscado, o -1 si no existe en el array.

1. Buscar "perro"

Busca en que posicion se encuentra "perro" dentro del array de animales.

2. Buscar numero 50

Verifica la posicion del numero 50, o obten -1 si cambias el array.

3. Ciudades con input

Busca en que posicion esta la ciudad que ingreses.
