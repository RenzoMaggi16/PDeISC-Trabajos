forEach()

Metodo forEach() — Iterar sin retorno

forEach() ejecuta una funcion por cada elemento del array. No devuelve un nuevo array ni modifica el original.

1. Saludos

Muestra un saludo por cada persona en el array.

2. Doble de cada numero

Muestra el doble de cada numero (no genera un array nuevo, solo lo imprime).

3. Objetos {nombre, edad}

Muestra los datos de cada persona almacenada como objeto en el array.
