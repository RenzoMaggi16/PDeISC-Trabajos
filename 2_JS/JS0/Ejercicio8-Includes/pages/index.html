Metodo includes() — Verificar existencia

includes() devuelve true si el elemento existe en el array, o false si no. Solo confirma presencia, no da la posicion.

1. Buscar "admin"

Verifica si el rol admin o superadmin existen en el array.

2. Colores con input

Verifica si el color que ingresaste existe en la lista.

3. Agregar numero si no existe

Evita duplicados usando includes() antes de agregar.
